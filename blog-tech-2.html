Element can sometimes act in very unamusing ways and can seem to not do what you want them to but with a bit of mastery in the position style we can turn into a true frontend hacker and gain incredible control over our web pages and where we position our content.

Postion Static

Position static is the default position, one doesn’t even need to add this to your css, Static positions are not affected by the top, bottom, left and right properties. Static position is fairly basic, static elements position them selves next to or underneath depending if they are text - inline or div blocks.

Position Relative

Position relative gets a bit more exiting having positon relative lets and object have a z-index, lets us stack elements ontop of one another. It can also be affected by the top, bottom, left and right properties. These properties are by defauklt set ot 0 but take a numerical value such as top: 100px which would place the element 100px away from the top in this case with a positive value it will move downwards on the page with a negative value it will move above where it was. Most importantly when a parent object has a position set to relative a child element can be positioned anywhere on the DOM without limitations and without affecting the position of other elements.

yellow - static

green - relative

Postion Absolute

Position absolute will initially position a element relative to the document’s body element position element is manipulate using the top, bottom, left and right properties. When it has a parent element set to relative the child with absolute will be positioned you guessed it relative to its parent making the top, bottom, left and right properties based of the parent. If its parent element doesn’t have a positon et to relative it will find the next closet element other wise it will use the body

yellow- parent - relative

green - absolute

Postion Fixed

Positon fixed works in a similar way to position absolute but is set relative to the document body always. If you hadn't noticed it is the little yellow box on the bottom left of the screen. This element is fixed to this position using the top, bottom, left and right properties and will remain in this position as we scroll thorugh the page. Position fixed is often used for navigation menus so as the user scrolls the nav menu remains in view and can always be interated with.
